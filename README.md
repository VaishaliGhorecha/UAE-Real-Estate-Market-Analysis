📊 UAE Real Estate Market Analysis

🏙️ Project Overview

This project analyzes the UAE Real Estate Market using data analytics and interactive dashboards.

The objective is to help stakeholders:

✔ Understand market performance
✔ Identify high-value investment areas
✔ Analyze property demand and occupancy trends
✔ Evaluate property price & size distribution across cities

The project uses data visualization and business insights to support data-driven investment decisions.

🧰 Tools & Technologies
📊 Microsoft Power BI – Data Visualization
📑 Microsoft Excel – Data Cleaning
📈 Data Analysis
📉 Business Intelligence

🎯 Business Objectives

The goal of this analysis is to answer key real estate questions:

• Which UAE cities generate the highest property value
• Which locations provide strong investment opportunities
• What is the occupancy vs vacancy trend
• Which property types are most in demand
• How property prices vary across cities
